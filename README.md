> *Senior Project (โครงงานทางวิชาการ)*

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

**สิ่งที่ทำ:**
- ✅ พัฒนาระบบรถบังคับผ่านเครือข่าย **Wi-Fi** โดยใช้ Raspberry Pi Zero 2 W
- ✅ ควบคุมมอเตอร์ด้วยภาษา **Python** และโตรไดเวอร์มอเตอร์ **L298N**

- ✅ พัฒนาการสื่อสารผ่านโปรโตคอล **TCP/IP** สำหรับการควบคุมแบบเรียลไทม์
